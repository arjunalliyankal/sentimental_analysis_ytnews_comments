The project can provide valuable insights into public opinion and understand the impact of news stories. Technology used :Google API Client, NLTK, Scikit-learn,pandas,numpy,matplotlib The software can be used to analyse comments from videos on platform like youtube. these videos contain news stories viewed by common people. The software scrapes the comments from these videos and then performs sentimental analysis on them. The software categories the emotions of people towards the news stories into three categories Positive Negative and Neutral
